## Features

- 300-frame image sequence animation driven by scroll
- Smooth scroll via Locomotive Scroll + GSAP ScrollTrigger
- Animated marquee text banners
- Responsive portfolio sections — About, Projects, Skills, Contact
- Modern typography with Syne + Space Grotesk + Space Mono
- Card spotlight hover effects
- Mobile-friendly native scroll fallback

## Tech Stack

| Layer | Tools |
|---|---|
| Animation | GSAP, ScrollTrigger, Locomotive Scroll |
| Fonts | Syne, Space Grotesk, Space Mono |
| Languages | HTML, CSS, JavaScript |
| Hosting | Vercel |

## Project Structure

```
├── index.html          # Main page
├── style.css           # Base styles
├── content-style.css   # Section & component styles
├── script.js           # Scroll animation logic
├── gsap.min.js
├── ScrollTrigger.min.js
├── locomotive-scroll.min.js
└── assets/
    └── images/
        └── bg_male/    # 300 PNG frames for scroll animation
```

## Run Locally

Just open `index.html` in a browser — no build step needed.
